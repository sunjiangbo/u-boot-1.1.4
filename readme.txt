
********************************************************************
版本号：   boot-3511dvr-V1.0.bin
修改日期： 2009.12.7
修改人：   任赋
基本内容： 内核引导程序
修改内容：

1.修改下载模式的控制，由原来的任意键进入下载模式改为按下空格键有效;
	涉及文件:main.c――增加针对空格键的判断
2.添加并启动软硬件看门狗功能;
	涉及文件:watchdog.c――实现软硬件看门狗功能；
		 board.c――增加初始化看门狗调用
		 cmd_mem.c、cmd_nvedit.c、console.c、interrupts.c、
	 	 cfi_flash.c、serial_pl011.c――添加清看门狗功能
		 hi3511v110.h――添加针对看门狗的宏控制
3.使能BOOT状态下三串口的输出和信息接收，以方便程序的下载；
	涉及文件:serial_pl011.c――修改添加针对三个串口的初始化和数据接收以及控制台串口的确认功能
		 hi3511v110.h――添加针对三串口功能的宏控制
4.添加启动时读环境变量错误自动复位功能；
	涉及文件：main.c――添加针对读环境变量错误的处理功能
5.添加自定义版本信息输出；
	涉及文件:hi3511v110.h――添加版本信息宏定义		 
6.实现在控制台下1分钟无命令输入启动内核的功能，防止在干扰情况下误进入控制台而无法启动系统；
	涉及文件:main.c――增加1分钟超时的启动系统处理功能
		 hi3511v110.h――添加延迟时间的宏定义
7.添加flash的写保护和下载程序时的flash自擦除功能，方便程序的下载。
	涉及文件:cfi_flash.c――实现下载程序时的flash的自擦除功能
		 hi3511v110.h――添加FLASH写保护的宏定义
8.关闭海思对系统资源的占用。
	涉及文件：lowlevel_init.c――去除系统对GPIO口的占用；
		  board.c――设置系统的复用功能
